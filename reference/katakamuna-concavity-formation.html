3. Phase A — Concavity Formation: 第4〜7首

Aperture → Projection → Directional Tension → Curvature

3.1 Plate 4 — Outer ApertureField

Plate_4 = C_home + ApertureArray(H_i, d_i)

第4首では、中心に成立済みの閉場 C_home があり、その周囲に方向の異なる複数の半円系 H_i が配置される。

これは単なる複数候補ではない。各 H_i は異なる開口方向 d_i を持つため、それぞれが局所的な観測フレームとして働く。

第4首は、単一正面を崩し、複数方向の開口 H_i を立てる外周開口場である。

3.2 Plate 5 — Outer Many-Glyph Projection Field

Plate_5 = C_home + X_internal + S[x] + P8-Ring + OuterManyGlyphField

第5首では、中央に C_home + X_internal + S[x] + P8-Ring が成立し、外周の多数 glyph 群が中央の八位相盤へ射影される。

Outer Many-Glyph Field → P8-Ring

第5首は、外周の多 glyph 場を中央 P8 Core へ射影する位相投影盤である。

3.3 Plate 6 — Outer Directional X-L Field

Plate_6 = P8 Core + OuterDirectional(X,L)

第6首では、第5首と同型の P8 Core を保持したまま、外周に X/L 系の方向性場が強まる。

第5首が投影なら、第6首は作用である。

第6首は、共有された P8 Core に外周 X/L 場が方向性をもって作用する盤面である。

3.4 Plate 7 — Outer Curvature-Tension Field

Plate_7 = X_warp + CurvatureTensionField

第7首では、第6首の方向性 X/L 作用が進み、直線的な裁定軸 X が曲率化・張力化する。

X-straight → X-directed tension → X-warp

第7首は、方向性を持った X/L 作用が曲率張力へ変わり、X-warp として凹形成を開始する盤面である。

4. Phase B — Re-Synchronization and Functional Differentiation: 第8〜9首

Curvature → P8 Re-synchronization → C/X composite differentiation

4.1 Plate 8 — P8 Core Re-Synchronization Field

Plate_8 = P8 Core + MixedOuterReSynchronizationField

第8首では、中央に P8 Core が明確に戻り、外周の H/C/X/L/P 系 operator が混合的に再配置される。

第7首で曲率化した場が、散逸せずに P8 中央へ再同期される盤面として読める。

第8首は、曲率化した外周場を P8 中央へ再同期させる統合盤面である。

4.2 Plate 9 — Distributed C/X Composite Units

Plate_9 = P8 Core + Distributed(C/X Composite Units)

第9首では、P8 Core を保持したまま、外周に C/X 系の複合単位が分化し始める。

これは第15首の機能層化へ向かう前段でもあり、第10首以降の帯形成の入口でもある。

第9首は、P8 Core を保持したまま、外周に C/X 複合単位を分布させ、場を機能分化へ移す移行首である。

5. Phase C — Band Formation and Maturation: 第10〜14首

Band Formation → Fold Event → Re-expansion → Saturation → Stabilization

第10〜14首では、外周 C/X/P 系の局所セル群が帯として組織化され、一度中心を折り込み、再展開し、飽和し、固定される。

5.1 Plate 10 — Peripheral C/X Cell Ring

Plate_10 = P8 Core + Peripheral(C/X Cell Ring)

第10首では、外周に C/X 系の局所セル群がリング状に同期し始める。

第10首は、P8 中央構造を保持したまま、外周 C/X セル群を環状同期させ、機能分化を帯形成へ進める盤面である。

5.2 Plate 11 — Fold Event / C_diamond

Plate_11 = C_diamond inside Peripheral Band
C_diamond = C + X_internal + Diamond_4 + P4

第11首では、中央がそれまでの P8 Core ではなく、C_diamond へ変化する。

これは帯形成の中で一拍だけ起きる折り込みイベントとして読める。

第11首は、外周 C/X セル帯を維持したまま、中央を P8 Core から C_diamond / P4 Foldable Cell へ変換し、帯形成中に折り畳み中心を発生させる盤面である。

5.3 Plate 12 — Re-expanded P8 Core

Plate_12 = P8 Core + DensePeripheral(C/X Band)

第12首では、中央が再び P8 Core へ戻る。第11首の C_diamond は恒常化ではなく、折り込みイベントとして働いたと見られる。

第12首は、第11首の C_diamond 折り込みを経たあと、中央を P8 Core へ再展開し、外周 C/X セル帯を高密度化・均質化する盤面である。

5.4 Plate 13 — Saturated C/X/P Band Field

Plate_13 = P8 Core + Saturated(C/X/P Band Field)

第13首では、折り込み後に再展開された帯が、外周で高密度に同期・飽和する。

第13首は、第11首の折り込みと第12首の再展開を経た P8 Core の周囲で、C/X/P 系の外周セル帯が高密度に同期・飽和する盤面である。

5.5 Plate 14 — Stabilized Peripheral C/X/P Band

Plate_14 = P8 Core + StabilizedPeripheral(C/X/P Band)

第14首では、P8 Core を保持したまま、外周 C/X/P セル帯がさらに均質化・圧縮・安定化される。

この段階で中心はまだ空いていない。むしろ、中心を空けるための外周保持力が完成しつつある。

第14首は、P8 中央構造を保持したまま、外周 C/X/P セル帯をさらに均質化・圧縮・安定化し、第15首の C_empty 化を可能にする直前段階の盤面である。

6. Phase D — Empty-Center Stabilization: 第15首

Plate_15 = C_empty + ConcentricFunctionalLayers

第15首では、中央構造が決定的に変化する。第4〜14首で繰り返し現れた P8 Core ではなく、中央には C_empty が置かれる。

C_empty = circle only
          no X
          no P-ring
          no internal articulation

外周には C/X/P 系の機能閉場が多層状に配置される。

重要なのは、C_empty が単なる欠如ではないという点である。第10〜14首で外周帯が成熟したため、中心は支えの役割から解放され、空になることが可能になる。

P8 Core supports field
→ peripheral band matures
→ center is released
→ C_empty appears

第15首は、これまでの C_home / P8 Core とは異なり、中央を X も P-ring も持たない C_empty として空け、その外周に C/X/P 系の機能閉場を多層配置することで、成熟した外周帯による空中心安定化を示す盤面である。
