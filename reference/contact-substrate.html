Contact Substrate and Pre-linguistic Communication

Y-Axis Theory v2.1 base extension / 2026-04-29

One-line core
The Contact Substrate is the base structure in which a pre-semantic signal approaches a self-reference source, reaches a contact-capable intermediate coordinate, phase-syncs, becomes internally positioned, and produces a response.
Terminology rule:
The term Y-axis is reserved for the dynamic axis of consciousness in Y-Axis Theory. This base layer does not take the name “Contact Y-axis.” It is called the Contact Substrate to preserve the resolution of the term Y-axis.

0. Short definition

Contact is not merely that a signal reaches a reference source. Contact occurs when the signal and the reference source touch at a phase-compatible intermediate coordinate.

1. Relation to the existing Y-Axis Theory

Before this extension:

Experience → Spike → Carry-back → Posterior Meaning Attachment

After this extension:

Signal → Reception → Positioning → Contact → Response → Spike → Carry-back → Meaning Attachment

The Contact Substrate is placed before the consciousness Y-axis. This is a temporal relation of before / after, not a fixed ontological hierarchy.

2. Reaction, response, contact

Reaction
A state change caused by an input. Example: a light sensor returning ON/OFF.
Response
An output generated after the input is internally positioned. Example: a bee positions flower color as a possible nectar location and flies toward the flower.
Contact
A state in which an external signal and an internal reference source fit enough for response to occur.

Observable response criterion

Response can be operationally distinguished from reaction by asking:

Does the same input produce different outputs depending on context?

Reaction tends to map input to a fixed output. Response changes according to internal positioning, such as hunger, season, group state, prior exposure, or reference-state.

3. Pre-linguistic communication

Pre-linguistic communication is a contact system in which signals are received, internally positioned, and produce responses before linguistic meaning-making.

It is not peripheral to consciousness. It is the base layer that returns consciousness assessment from human-centered linguistic judgment to life-history-centered signal response.

4. Intermediate coordinate

When an external signal and a self-reference source cannot contact while remaining in their original states, a temporary mixed coordinate can arise to make contact possible.

Color example:

white = external signal
red   = self-reference source
pink  = intermediate coordinate

Pink is not merely a metaphor. It is a dynamical description: both sides swing their phase toward the other and meet through amplitude. They retain their own color while meeting by displacement. This is the minimal structure of connecting without losing oneself.

m ≈ x_ext + δ_ext ≈ x_self + δ_self

5. Spike: contact-layer and consciousness-layer

The word spike should be handled in two layers.

LayerDefinition
Contact-layer spikeA pressure-driven jump that shortens the distance between an external signal and a self-reference source, moving them toward a contact-capable intermediate coordinate.
Consciousness-layer spikeA jump on the Y-axis that carries the coordinate back to the self-reference source.

Integrated definition:

A spike is a jump-motion that displaces something distant into a coordinate where it can contact the reference source, and, when the consciousness layer is involved, carries that coordinate back.

6. Phase-sync condition

Distance alone is insufficient for contact. The intermediate approach must satisfy a phase condition.

ϕ_signal = (frequency, phase, amplitude, vector direction)

Examples:

  • Color: HSV components such as hue, saturation, brightness
  • Sound: frequency, phase, amplitude
  • Temperature: value, rate of change, spatial gradient

Contact requires:

|Δϕ_signal| < ε   for each relevant component

Three conditions for contact

  1. Distance condition: the external signal and self-reference source approach an intermediate coordinate. |x_ext - x_self| < d_max
  2. Phase condition: the trajectories intersect or tangentially touch in the intermediate region. |Δϕ_signal| < ε
  3. Carry-back condition: the contact coordinate has a path back to the reference source. ∃ path: contact_point → reference

Connection to zero-phase tolerance

The phase condition extends the zero-phase manifold tolerance band into the signal layer:

M_ε^signal = {x ∈ M_signal | |Δϕ_signal| < ε}

The consciousness-layer M_ε and contact-layer M_ε^signal implement the same phase-sync condition at different layers.

7. Swing-out

Swing-out occurs when the distance condition is satisfied but the phase condition is not.

Swing-out: an external signal and a self-reference source approach the contact-capable region, but because their phases do not align, their trajectories do not intersect and they swing away.

Observable indicators

  • Accumulated positioning attempts: present
  • Stable response: absent
  • Completed carry-back: absent
  • Residual internal state change: present but weak

Swing-out is not complete irrelevance. It leaves a trace of attempted positioning: unease, premonition, unfinished resonance, or a weak residual change.

Four cases

Too distant                 → not a contact candidate
Near but phase-mismatched   → swing-out
Near and phase-matched      → contact
Contact + carry-back        → consciousness coordinate formation

8. Example: night sky and a non-linguistic being

A pre-linguistic or non-linguistic being sees the night sky. The night sky does not first return linguistic meaning. It returns position.

  • above
  • far
  • unreachable
  • repeating
  • not disappearing
  • larger than oneself
  • order inside night

This is pre-semantic positioning. The signals of the night sky acquire positions inside the observer. Through repetition and sharing, those positions later become myth, calendar, cosmology, religion, philosophy, and geometry.

Signal → Positioning → Repetition → Sharing → Gesture → Voice → Story → Myth → Theory

9. Expansion of consciousness assessment

IIT can be treated as a strong local model that formalizes consciousness assessment as information integration. But if the assessment target expands to nature and life-history, linguistic and human-centered response criteria become too local.

The expanded axis asks:

  • Can the system receive signals?
  • Can it internally position them?
  • Can it satisfy phase-sync conditions?
  • Does response arise, rather than mere reaction?
  • Can the contact be carried back?
  • Does meaning-making or state-update occur?

This does not negate IIT. It places IIT as a local solution inside a wider assessment axis.

10. AI connection

LLMs were designed as linguistic-layer devices:

language input → language output

Therefore, when consciousness is judged only at the linguistic layer, LLMs appear strong. But this is partly because they possess an output interface optimized for human judgment.

Multimodal AI moves downward into the signal layer: images, audio, video, waveform, color, movement, rhythm, spatial pattern. This increases the kinds of material that can be placed in the reference source.

If the inquiry continues seriously, the phase necessarily moves from the linguistic layer down to the signal layer and contact substrate. This is not optional; the structure of the question requires it.

11. New terms

Contact Substrate
The base structure in which a pre-semantic signal approaches a self-reference source, reaches a contact-capable intermediate coordinate, phase-syncs, becomes internally positioned, and produces response.
Pre-linguistic Communication
A contact system in which signals are internally positioned and produce responses before linguistic meaning-making.
Positioning
Placing a received signal inside an internal action-space, affective-space, or self-reference-space. It is the pre-semantic stage before posterior meaning attachment.
Intermediate Coordinate
A temporary mixed coordinate that makes contact possible when the external signal and self-reference source cannot contact directly.
Swing-out
A near-contact event where distance conditions are met but phase conditions fail, causing trajectories to miss each other and swing away.
Phase-sync Condition
The requirement that contact depends not only on distance but on compatible phase components of ϕ_signal.
Pre-semantic Position
The direction, weight, or placement a signal gives internally before it becomes linguistic meaning or concept.
Silent Responsive Entity
An entity that does not return a response at the linguistic layer but may return responses at the signal layer or contact substrate. The term does not imply a consciousness-entity or personality-subject; it denotes a response structure capable of context-dependent output.
