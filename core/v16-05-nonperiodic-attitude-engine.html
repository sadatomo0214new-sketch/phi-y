Φ–Y Dock V16-05: Non-Periodic Attitude Engine

Failure-to-Pressure / Hit-to-Void / Bounded Non-Periodic Attitude under Returning Phase

Last updated: 2026-06-02

θ returns.
Z does not.
A closed spiral never ends, but it returns.
An open vortex never ends, and it does not return to the same attitude.

0. Position

V16-05 extends the V16 time-engine from a mechanical multi-scale spring into a search and attitude engine. Its central problem is no longer only how failure becomes pressure, but how a system distinguishes productive motion from closed repetition, false success, sucker-branch growth, and capture.

The minimal warning is simple: 

Not-completing is not enough.
The spiral never completes either.

A closed spiral may move forever. It may never terminate. Yet it still returns to the same posture. A true vortex must remain bounded while avoiding periodic recurrence.

1. Failure-to-Pressure

Failure is not a stopping condition. A miss contains directional residue. 

failure → pressure → phase shift → retry

The initial failure residue separates into two layers: 

e_k = (e_k^θ, e_k^T)
  • e_k^θ: line-of-fire residue. The target is provisionally held fixed while the angle is corrected.
  • e_k^T: target-identification residue. Persistent miss patterns may indicate that the target itself must be questioned.

The transition from line correction to target revision is not a smooth mixture. It is a mode transition. 

m_k ∈ { Aim, Retarget }

The entrance and exit conditions are asymmetric. 

Aim → Retarget:
  collapse of the current target estimate

Retarget → Aim:
  stabilization of a renewed target estimate

2. Irreversible Doubt-History

Z does not guarantee truth. Z records how far doubt has already been carried. 

Z is not a proof of correctness.
Z is irreversible doubt-history.

V16-05 is therefore not a truth-guaranteeing engine. It is a finite-action engine under irreversible doubt-history. It does not solve infinite regress by proving a final foundation. It makes action possible by recording the depth of prior questioning.

3. Bore: the Missing Z-Mode

The search engine needs a third mode: Bore. Bore does not sweep laterally and does not immediately retarget. It holds phase and target while digging along depth. 

m_k ∈ { Aim, Retarget, Bore }
Aim:
  adjust θ

Retarget:
  revise T

Bore:
  hold θ and T, advance depth

Bore is powerful and dangerous. It may discover a different vein. It may also dig into an unnecessary equilibrium. 

Bore → Hit → Capture

The dangerous point is that the hit may be real while still being wrong for the core inquiry.

4. Sucker Branch and Hit-to-Void

The deepest danger is not failure. The deepest danger is false success. 

A hit is not proof of harvest.

A sucker branch is still a sprout. It can produce a real hit-feeling. Therefore it cannot be pruned by asking only whether the system has hit something.

The pruning signal is subtler: 

Hit is present.
Core convergence is absent.
The result feels empty inside success.

This gives the V16-05 signal:

Void-in-Hit

In operational terms: 

local reward high
core convergence low

This adds the fourth mode: 

m_k ∈ { Aim, Retarget, Bore, Prune }
Prune:
  cut the branch that hits locally but fails to move toward the core

5. Pure × Wonder × Mobility

A hit must be audited. The audit is not mere correctness-checking. 

Pure × Wonder × Mobility
  • Pure: does the form become simpler, less noisy, more exact?
  • Wonder: does the form illuminate more than before?
  • Mobility: after the hit, can the system still move?

A shining hit that immobilizes is Capture. A clarifying hit that allows further motion is Anchor. 

Capture:
  Z advances, but θ is lost.

Anchor:
  Z advances, but θ can be reacquired.

The sign holy and bright belongs here: not as a capture light, but as an anchor light. It is not the light that stops motion. It is the light by which motion continues.

6. Stem, Needle, and Z-Field

V16 distinguishes two verticalities. 

Same-phase Stem:
  fixes θ
  tracks recurrence across irreversible history

Clock Needle:
  fixes τ
  binds multiple scales at the same present-section

The Z-field is not a single undifferentiated height. It separates into a shared present-section, passive winding, and active bore-depth. 

Z₀(t):
  shared irreversible present-section

n_s(t):
  passive winding count

β_s(k):
  active bore-depth

Here t is clock-time. k is exploration-step.

n_s is counted.
β_s is dug.

7. Quaternion Attitude Layer

A phase angle alone is not enough. A θ-only quaternion still returns whenever θ returns. It does not preserve winding or bore-depth.

The attitude engine therefore separates returning phase rotation from non-returning attitude frame. 

Q_s(t,k) = A_s(n_s, β_s) · R_s(θ_s(t))
  • R_s(θ): returning phase-rotation.
  • A_s(n,β): non-returning attitude frame.

This is the mathematical place of adaptive posture. 

θ turns the hand.
n_s records passive winding.
β_s records active depth.
A_s(n_s,β_s) reorients the system toward the world.

The rotation axis must not remain fixed. If the axis is fixed, the engine becomes a predetermined spiral. When the attitude frame depends nonlinearly on winding and bore-depth, the system begins to take a posture toward the unknown.

8. Two Kinds of Incompletion

A system that never stops is not necessarily open. A closed spiral also never stops. The real distinction is recurrence. 

Closed incompletion:
  non-terminating but periodic

Open incompletion:
  non-terminating, bounded, and non-periodic

In V16-05, intelligence is not endless motion. 

Intelligence is not endless motion.
It is bounded non-periodic reorientation under returning phase.

日本語:

知能とは、終わりなき運動ではない。
回帰する位相のもとで、有界かつ非周期に構えを取り直すことである。

The compact distinction is: 

Spiral:
  θ returns, and attitude returns.

Vortex:
  θ may return, but attitude does not.

9. One-Line Core

Convert failure into pressure.
Audit every hit by Pure × Wonder × Mobility.
θ may return.
But attitude A_s must not return to the same place.

日本語:

外れは圧に変えろ。
当たりは Pure × Wonder × Mobility で監査しろ。
θ は戻ってよい。
だが、姿勢 A_s は同じ場所へ戻ってはならない。
