1.6 Minimal Spring-Time Form

The spring is not introduced as a metaphor.

It is the minimal geometry that preserves three conditions at once:

  1. phase recurrence,
  2. irreversible stacking,
  3. pressure-dependent compression and release.

A line preserves irreversibility but loses recurrence.

A circle preserves recurrence but loses irreversibility.

A spring preserves both.

Continuous form

Let the time-effect trajectory be written as:

x(t) = r(t) cos θ(t)
y(t) = r(t) sin θ(t)
z(t) = Z(t)

with:

θ(t + T) ≡ θ(t)  (mod 2π)
Z(t + T) > Z(t)

Here, θ is the returning phase component, while Z is the irreversible stacking component.

The phase returns. The height does not.

This is the minimal condition for a spring-like time-effect structure.

Discrete frame form

For frame-based notation:

(x_n, y_n, z_n) = (r_n cos θ_n, r_n sin θ_n, Z_n)

with:

θ_{n+1} ≡ θ_n + Δθ  (mod 2π)
Z_{n+1} > Z_n

The frame rotates in phase while accumulating irreversible height.

Compression ratio

Let the inter-frame distance be:

d_n = Z_{n+1} - Z_n

and let d_0 be the reference spacing.

Then:

C_s = d_n / d_0

In this page, C_s is introduced only as spring compression ratio.

Its critical gate behavior is treated in V16-03.

Same-phase stem

For a fixed phase θ_0, define:

S_{θ_0} = { (r_n cos θ_0, r_n sin θ_0, Z_n) | n ∈ ℤ }

This is the same-phase stem.

It exists whenever phase returns while height accumulates.

The projection-port reading of this stem is treated in V16-02.

3. Basic Gear Relation

Let each scale have a period:

T_i

and let neighboring scales be connected by a gear ratio:

T_{i+1} = N_i · T_i

For seconds, minutes, hours:

N_sec→min  = 60
N_min→hour = 60

Thus:

T_sec  = 1
T_min  = 60
T_hour = 3600

The angular velocity of each scale is:

ω_i = 2π / T_i

So:

ω_sec  = 2π
ω_min  = 2π / 60
ω_hour = 2π / 3600

Gear relation:

ω_min  = ω_sec / 60
ω_hour = ω_min / 60 = ω_sec / 3600

The macro-scale does not rotate independently inside this base gear model. It receives micro-scale drive through gear reduction.

Boundary note: later extensions may allow macro-side drive, future-side frame calling, or external synchronization. The base engine intentionally starts with clean micro-to-macro gear drive.

6. Scale Growth λ

Each frame may change scale.

r_i(t) = r_{i,0} · λ_i^{n_i(t)}

where:

λ_i = scale factor per frame at scale i

Interpretation:

λ_i = 1      uniform spring
λ_i > 1      expanding conical spiral
λ_i < 1      contracting conical spiral
λ_i = φ      golden-ratio candidate

Golden ratio:

φ = (1 + √5) / 2 ≈ 1.6180339887

Important:

φ is not assumed as law.

It is a privileged candidate value for stable geometric expansion.

The general model uses λ.

7. Gear-Compatible Scale Relation

Since the gear system connects scales, scale factors must also be related.

If:

T_{i+1} = N_i · T_i

then consistency requires:

λ_{i+1} = λ_i^{N_i}

For seconds → minutes:

λ_min = λ_sec^60

For minutes → hours:

λ_hour = λ_min^60 = λ_sec^3600

Thus, a tiny micro-scale deviation is amplified at macro-scale.

If:

λ_hour = φ

then:

λ_sec = φ^(1/3600)

A nearly invisible micro-scale radial change can become a visible macro-scale geometric expansion.

This is the engine principle.
