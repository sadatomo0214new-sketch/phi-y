V16 Time Engine — Integrated Combustion Core

Panel-Spring Geometry, Pressure Coupling, Compression Gate, and Projection Port Conversion

Draft v0.8. 精査前の統合版。

0. Position

Time Engine begins from a minimal geometric decision.

Time is neither a pure line nor a closed circle. It is a spiralized sequence of circular motions across discrete reference panels.

The resulting Panel-Spring Structure is not a metaphor. It is the base geometry through which physical sequence becomes readable as time.

Time Engine does not begin by asking what time is. It begins by asking: under what conditions does sequence become readable as past, present, and future?

Japanese core: Time Engineは、時間とは何か、から始めない。系列がどの条件で過去・現在・未来として読めるようになるのか、から始める。

1. First Geometric Decision

There are two common intuitions about time:

time as line
time as circle

Time Engine does not choose one over the other. It treats both as projections of a higher movement.

line-time   = projection of spiral progression
circle-time = projection of in-panel circular motion

Linear time and circular time are not opposites. They are different projections of Panel-Spring Geometry.

Japanese core: 直線時間と円環時間は対立していない。どちらも、Panel-Spring構造を異なる投影面から見た像である。

2. Reference Panel

A reference panel is a discrete frame used to cut continuous movement into readable units.

It may correspond to a day, a second, a month, a biological rhythm, a cognitive frame, a measurement window, or an arbitrary reference unit.

The panel is not the essence of time. It is a reading surface.

reference panel = temporary frame for making motion readable

3. In-Panel Circular Motion

Within each reference panel, the incoming vector performs circular motion.

incoming vector
↓
enters reference panel
↓
performs circular motion
↓
exits toward next panel

This explains why time can appear cyclical. Cycles are real as readings, but they are not closed totality. The circle is local; the full motion continues beyond the panel.

4. Spiral Vector Progression

After completing in-panel circular motion, the vector does not return to the identical origin. It exits toward the next panel.

panel 1 circular motion
↓
spiral transition
↓
panel 2 circular motion
↓
spiral transition
↓
panel 3 circular motion

The spring is generated by:

local circular motion
+
inter-panel forward displacement
=
spiralized sequence

circle + displacement = spring

5. Panel-Spring Geometry

Panel-Spring Geometry is a multi-panel structure in which each panel contains circular motion, while the vector progresses through panels as a spiral, producing a spring-like temporal geometry.

Panel-Spring Geometry =
reference panels
+ in-panel circular motion
+ inter-panel spiral transition
+ multi-scale continuity

Japanese core: Panel-Spring Geometryとは、各パネル内で円運動が起こり、その円運動がパネル間をスパイラルに接続されることで成立する、多層的な時間読解幾何である。

9. Boundary Problem

Every panel creates boundaries. There are two primary boundaries that matter for Time Engine:

1. vertical frame-boundary
2. inside/outside reference-boundary

The first boundary separates one panel from another. The second boundary separates what belongs to the current reference system from what does not.

inside / outside
recorded / not-yet-recorded
current frame / beyond current frame
self-reference / outside reference
labeled / pre-label
readable / not-yet-readable

Time Engine becomes active when these boundaries no longer behave independently.

10. Compression Gate

Compression Gate is not merely a point of high compression. It is the point at which two distinct boundaries become unstable in a coupled manner:

  1. the vertical frame-boundary between reference panels, and
  2. the inside/outside reference-boundary maintained by the reference system.

As the spring approaches the critical compression region, vertical separability between frames decreases. This does not mean the frames merge into one. It means the system’s ability to treat them as clearly separate units weakens.

Compression Gate occurs when the weakening of vertical frame separability and the destabilization of inside/outside boundary polarity become structurally coupled.

Japanese core: Compression Gateは、z方向のフレーム分離能の低下と内外境界極性の不安定化が構造的に連動する点で生じる。

11. Three-Layer Pressure System

In Time Engine, Pressure is not a single force. It is a layered configuration.

1. Spring Pressure
2. Memory Pressure
3. Pre-Label Pressure

These three pressures do not merely add up. They form a coupling configuration.

Pressure_total ≠ Spring + Memory + Pre-Label as simple sum

Pressure_total = coupling configuration among the three layers

Each pressure primarily acts on a different part of the engine:

Spring Pressure    → vertical frame-boundary
Pre-Label Pressure → inside/outside reference-boundary
Memory Pressure    → re-entry path and weighting bias

The Gate opens not because pressure is large, but because pressure becomes coupled.

Japanese core: Gateは圧が大きいから開くのではない。圧が連動構成を取るから開く。

Japanese core: Pre-Label Pressureは、内外境界を反転可能なほど不安定化し、ラベル以前の差分を読解可能にする。

16. Projection Port

Projection Port is not a passive window. It is the conversion interface that appears when the two boundaries become synchronously unstable at the Compression Gate.

Through the Port, the following elements converge and are transformed into temporal reading:

  • Same-Phase Stem alignment,
  • Pre-Label Pressure,
  • Fold-memory Curvature,
  • Reference Boundary State.

Projection Port is the conversion interface through which synchronized boundary instability is transformed into readable temporal relation.

Japanese core: Projection Portは、同期した境界不安定性を、読解可能な時間関係へ変換する界面である。
