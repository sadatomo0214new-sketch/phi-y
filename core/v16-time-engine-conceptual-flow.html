7. Runtime Pseudocode

function TimeEngine(sequence, reference_frame):

    panel_structure = build_panel_spring_geometry(sequence, reference_frame)

    stem = detect_same_phase_stem(
        phase_direction = θ₀,
        height = z,
        radial_scale = r
    )

    spring_pressure = compute_spring_pressure(panel_structure)
    memory_pressure = compute_memory_pressure(Fold_memory)
    prelabel_pressure = detect_prelabel_pressure(reference_frame)

    pressure_config = couple_pressures(
        spring_pressure,
        memory_pressure,
        prelabel_pressure
    )

    boundary_state = evaluate_boundary_state(reference_frame)

    gate_ready = check_compression_gate(
        pressure_config,
        vertical_frame_boundary,
        inside_outside_boundary
    )

    if not gate_ready:
        return ordinary_sequence_reading(sequence)

    port = open_projection_port(
        gate_condition = gate_ready,
        stem = stem,
        boundary_state = boundary_state
    )

    aligned = Align(θ₀, z, r)

    weighted = Weight(
        aligned,
        fold_memory = F,
        prelabel_pressure = Π
    )

    temporal_reading = Stabilize(
        weighted,
        boundary_state = B
    )

    return temporal_reading

Time is not produced by sequence alone. Time becomes readable when layered pressure couples boundaries, opens the Compression Gate, activates Projection Port conversion, and stabilizes phase-aligned pressure-memory relations as past, present, and future.
