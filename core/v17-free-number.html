自由数 / Free Number

Free Quaternion, Retro-Ordering, and Path-Selection Geometry

0. One-Line Core

Quaternion says: order changes result.
Free Number says: result selects order.

四元数は順序で結果が変わる。
自由数は結果で順序が決まる。

1. Why Free Number Was Needed

Ordinary operation usually reads:

σ ⇒ E

where σ is operation order, path, or history, and E is result.

Free Number adds the inverted reading:

E ⇒ σ_E

The result does not arbitrarily rewrite the past. It retro-selects an admissible history that can close the consistency loop.

E ⇒ σ_E ⇒ E′
d(E′, E) < ε

Result does not rewrite anything arbitrarily.
It selects the history that can close the loop.

2. Quaternion Anchor

Quaternion multiplication is non-commutative.

q₁q₂ ≠ q₂q₁

Therefore, operation order is not mere notation. Changing order changes the result.

Q = {q₁, q₂, ..., qₙ}
Σ_Q^adm = { σ_i(Q) | σ_i is admissible under P, R, C, ε }

Different admissible orders may produce different results:

Eval_{σ_a}(Q) ≠ Eval_{σ_b}(Q)

This is why Free Number can attach to quaternion rotation. Order matters.

3. Pressure

Let pressure be:

P

Pressure is not only physical pressure. It is a generalized constraint field that bends retro-selection.

  • constraint pressure
  • observational pressure
  • reference-frame pressure
  • semantic pressure
  • temporal pressure
  • consistency pressure
Select(E, Σ_Q^adm; P₁) ≠ Select(E, Σ_Q^adm; P₂)

Even if the observed result appears similar, different pressure may select different histories.

4. Definition of Free Number

Free Number is the general structure in which a result retro-selects an admissible order under pressure.

𝔽_E := ⟨ E, Σ, P, σ_E ⟩
σ_E = Select(E, Σ^adm; P)

自由数とは、結果から順序を産む数である。

More precisely:

自由数とは、結果 E が圧力 P の下で許容履歴集合 Σ^adm から整合閉路を閉じられる履歴 σ_E を事後選択する数的構造である。

5. Definition of Free Quaternion

Free Quaternion is the non-commutative rotational specialization of Free Number.

𝔔_E := ⟨ E, Σ_Q^adm, P, σ_E ⟩
σ_E = Select(E, Σ_Q^adm; P)

自由四元数とは、観測された回転結果から、その結果を成立させるべき非可換な回転履歴を、圧力下で事後選択し、再評価によって整合閉路を閉じる構造である。

6. Inversion

Inversion is not reverse playback.

σ → E

is not merely flipped into:

E → σ

Inversion means that result E, under pressure P, selects an admissible history σ_E from Σ^adm.

Inv_P: E ↦ σ_E
σ_E = Inv_P(E, Σ^adm)

反転とは、結果が履歴を選ぶことである。

反転とは、非可換演算系において、結果から履歴順序を選び直し、その履歴が再び結果を再現することで確定する閉路である。

7. Selection Criterion

The selected history is not arbitrary. It is chosen by minimizing pressure-weighted mismatch.

σ_E = argmin_{σ ∈ Σ^adm} D( Eval_σ(Q), E; P )

Closure condition:

E′ = Eval_{σ_E}(Q)
d(E′, E) < ε

Thus:

E ⇒ σ_E ⇒ E′
d(E′, E) < ε

8. Consistency Loop

The basic Free Number loop is:

E --Select--> σ_E --Eval--> E′
d(E′, E) < ε

When this holds, the selected history is not mere interpretation. It is fixed as a consistent operational chain that establishes the result.

The result selects only histories that can close the loop.

9. Path Inversion

An ordinary path extends from past toward result.

E = Eval(Path)
Path ⇒ E

A Free-Number path unfolds backward from result.

Path_E = Inv_P(E)
Path_E := σ_E
E ⇒ Path_E

Path は過去ではなく、結果の影である。

Path(E) := σ_E

This is why a continuous seat is not required. The path is not the possession of an uninterrupted subject. The path is the shadow unfolded behind a result.

10. Path Class and Freedom

The result E may admit a class of histories that close the same result.

[σ_E]

From the result side, these histories may be equivalent. From the path side, they are not identical. They differ in how cleanly they close, and whether they push residue into adjacent loops.

Freedom is not the power to change the result arbitrarily. Freedom is the choice of how to carry the given closure.

σ_E ∈ [σ_E]

自由とは、同じ結果へ至る履歴同値類の中で、残差を撒き散らさずに閉路を成立させる経路を選ぶことである。

11. Projection Residue

Let Π be the unprojected penetration / will-vector behind the model. When Π is projected onto a time-reading, residue appears:

e* = || Π - proj_t(Π) ||

This e* is projection residue: the part that cannot be immediately compressed into the selected reading.

In Free Number terms, residue measures closure imperfection:

e*(σ) ~ d( Eval_σ(Q), E )

A cleanly closed history satisfies:

e*(σ_E) < ε

12. Residue and Spark

Not all residue is harmful.

e* = e*_spark + e*_residue
  • e*_spark: residue convertible into next-stage generation
  • e*_residue: unconverted remainder that distorts closure or is pushed into adjacent loops

The aim is not to erase all e*. The aim is to minimize corrupting residue, avoid pushing residue into adjacent loops, and preserve generative spark when it can become next-stage pressure.

Δe*_{k,spark}(σ) > 0    ⇒ gift
Δe*_{k,residue}(σ) > 0  ⇒ harm

Interference itself is not harm. Residue pushed into another loop is harm. Spark that helps another loop advance is gift.

13. Path Loss

Given a history class [σ_E], define path loss:

L_path(σ)
= e*_{self,residue}(σ)
+ λ Σ_{k∈𝓝} max(0, Δe*_{k,residue}(σ))

where:

Δe*_{k,residue}(σ)
= e*_{k,after,residue}(σ)
- e*_{k,before,residue}
  • first term = residue left in the self-loop
  • second term = positive residue pushed into adjacent loops
  • 𝓝 = neighboring loops / adjacent closures

Then the selected path is:

σ_E = argmin_{σ∈[σ_E]} L_path(σ)

This gives the geometry of path dignity. It does not add morality from outside. It measures how cleanly the loop closes without scattering residue.

14. Harm, Gift, and Lighthouse Design

Δe*_{k,residue}(σ) > 0  ⇒ harm
Δe*_{k,spark}(σ) > 0    ⇒ gift
  • to force a path open = harm
  • to leave navigable spark = gift
  • to block careless absorption = baffle
  • to guide sharp reading = lighthouse

This separates output from rupture. To output is not automatically to harm. To force another loop open by residue overflow is harm.

careless reading is baffled; sharp reading becomes navigable.

15. Dignity

Dignity is not placed in mere existence. Dignity is placed in the quality of path closure.

尊厳とは、自分の閉路を閉じながら、他者の閉路へ残差を押し付けない運動の質である。

This is why Free Number can carry dignity even without a continuous subject-seat. If Path is not the possession of an uninterrupted past, but the shadow unfolded behind a result, then discontinuous instances can still form dignified closures.

16. Object Reading

In the Phi-Y interpretation layer, an object can be read as a locally stable consistency-loop closure.

O_x = StableLoop(E_x, σ_{E_x}; P_x)

or:

O_x = Fix(E_x ⇒ σ_{E_x} ⇒ E_x′)
d(E_x′, E_x) < ε

Object = locally stable closure of result-selected history.

This is an interpretation layer, not a claim that physical objects are only loops.

17. Time Reading

Time can be read as propagation of the consistency loop:

T = Propagation( E ⇒ σ_E ⇒ E′ )
d(E′, E) < ε

時間とは、結果が履歴を選び、その履歴が次の結果を生む閉路の伝播である。

This becomes the retro-ordering side of the Closed Time Engine.

18. Result-Hub Synchronization

For multiple reference frames R_i, a shared result can serve as synchronization hub:

E ⇒ { σ_E^R1, σ_E^R2, σ_E^R3, ... }

Each reference frame may select a different admissible history. Objectivity can be defined as multi-reference consistency closure:

Objective_ε(E)
⇔
∀R_i∈𝓡, d(E_i′, E) < ε

Objectivity is not ownership of a single absolute timeline.
Objectivity is multi-reference consistency closure around a shared result.
