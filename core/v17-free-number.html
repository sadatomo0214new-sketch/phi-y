自由数 / Free Number

Standalone Signboard

0. 主題

自由数とは、結果から順序を選ぶ数である。

四元数は順序が結果を決める。自由数はその向きを反転させ、結果が順序を決める。これは比喩ではなく、非可換性という同じ一つの足場の上で定義される。

四元数は順序で結果が変わる。
自由数は結果で順序が決まる。

結果は過去を勝手に書き換えない。閉路を閉じられる履歴を選ぶ。

1. 中核命題

四元数回転の本質は非可換性にある。

q₁ q₂ ≠ q₂ q₁

順序が変われば結果が変わる。ゆえに順序は表記ではなく演算実在である。

通常の読み:

σ ⇒ E      （順序が結果を産む）

自由数の反転:

E ⇒ σ_E    （結果が順序を選ぶ）

整合閉路:

E ⇒ σ_E ⇒ E′,   E′ ≈ E

2. 用語

2.1 回転列

Q = {q₁, q₂, ..., qₙ}
Σ_Q = {σ_i(Q)}        可能な演算順序の集合

非可換ゆえ:

Eval_{σ_a}(Q) ≠ Eval_{σ_b}(Q)

2.2 許容順序集合

自由数は無限の空想空間から選ばない。許容順序集合から選ぶ。

Σ_Q^adm = { σ_i(Q) | σ_i は P, R, C の下で許容される }

P = 圧力 / 一般化された制約場
R = 参照枠
C = 整合条件

これが反転を恣意的な後付け物語にさせない。

2.3 結果

E

単なる出力値ではなく、「そこに到達したという観測事実」。

2.4 圧力 P

P

圧力とは、許容履歴集合の中からどの履歴が選ばれるかを曲げる条件。

3. 自由四元数の定義

自由四元数 𝔔_E とは、観測された結果 E から、その結果を成立させる非可換な回転順序 σ_E を、圧力 P の下で事後選択し、再評価で確定する構造。

𝔔_E := ⟨E, Σ_Q^adm, P, σ_E⟩
σ_E = Select(E, Σ_Q^adm; P)

4. 選択基準

選択は恣意的でない。許容集合の中で、結果との不一致を最小化する履歴が選ばれる。

Σ_Q^adm(P,R,C) = { σ_i(Q) | σ_i は P, R, C の下で許容 }
σ_E = argmin_{σ ∈ Σ_Q^adm(P,R,C)} D( Eval_σ(Q), E )

P は候補集合 Σ_Q^adm を切る。D は候補の中で結果との不一致を測る。両者は役割を分けている。P は不一致関数に重みとして入らない。

自由数が主張しないこと:

どんな結果もどんな履歴を選べる   ← これは言わない

自由数が主張すること:

結果は、圧力によって画定された許容集合の中から、閉路を閉じられる履歴を選ぶ

5. 反転 / Inversion

Inv_P: E ↦ σ_E
σ_E = Inv_P(E, Σ_Q^adm)

反転は単純な逆再生ではない。σ → E を E → σ に向き変えするのではなく、結果 E が許容集合の中から、自身を成立させる履歴 σ_E を選ぶこと。

反転とは、結果が履歴を選ぶことである。

反転とは、非可換演算系において、結果から履歴順序を選び直し、
その履歴が再び結果を再現することで確定する閉路である。

6. 整合閉路 / Consistency Loop

選ばれた履歴は、再評価で結果を許容内に再現できねばならない。

E --Inv_P--> σ_E
σ_E --Eval--> E′
d(E′, E) < ε

短く:

E ⇒ σ_E ⇒ E′,   d(E′, E) < ε

この閉路が成立したとき、履歴は単なる解釈ではなく、結果を成立させる整合的回転連鎖として固定される。

§A. 残差と棘 — Residual and Thorn

整合閉路は完全には閉じない。閉じ残しが残る。この閉じ残しを残差と呼ぶ。

e* := d(E′, E)

e* は閉路が結果をどれだけ再現しきれなかったかの量である。

  • e* < ε のとき、閉路は閉じたとみなされ、履歴は固定される。
  • e* ≥ ε のとき、閉路は閉じきらず、残差が顕在化する。

この顕在化した残差が 棘 (Thorn) である。

棘とは、結果が選んだ履歴で再現しきれなかった残差である。

棘は欠陥ではない。閉じきらない e* があるからこそ、閉路は次の結果へ接続される。完全に閉じた閉路はその場で停止する。残差が運動を次へ送る。

e* = 0       完全閉路 / 停止
0 < e* < ε   安定した閉路 / 物体
e* ≥ ε       棘の顕在化 / 次の結果への射出

ε は閉路をどこまで「閉じた」とみなすかの許容解像度である。ε を大きく取れば閉路は緩く閉じ、小さく取れば厳しく閉じる。

7. Path の反転

通常の Path は過去から結果へ伸びる。

E = Eval(Path)
Path ⇒ E

自由数的 Path は結果から背後へ展開される。

Path_E = Inv_P(E)
Path_E := σ_E
E ⇒ Path_E

Path は過去ではなく、結果の影である。

§B. 物体の再定義 — 同値類による安定

v0.1:

O_x = Collision(Σ_x; P_x)
物体とは、順序自由度が圧力によって折り畳まれた痕跡である。

整合閉路として強化する。ただし、結果を再現する履歴は一本とは限らない。

許容集合の中に、結果 E_x を許容内 (e* < ε) で再現する履歴が複数あるとき、それらは一つの 同値類 をなす。

[σ_{E_x}] := { σ ∈ Σ_Q^adm | d(Eval_σ(Q), E_x) < ε }

物体は、この同値類が局所安定した結節点である。

O_x := StableLoop( E_x, [σ_{E_x}]; P_x )

物体とは、可能順序の衝突が、結果—履歴—結果の閉路として
局所安定した同値類である。

これが重要なのは、安定性の根拠が単数か複数かで変わるからである。

  • 一本の履歴だけで支えられた物体は、わずかな摂動で閉路が破れ、即座に崩れる。
  • 複数の履歴の束、つまり同値類で支えられた物体は、一本が破れても他が支え、安定する。

退化、すなわち複数の履歴が等しく結果を再現することは欠陥ではない。退化こそが物体を物体たらしめる。

8. 時間の再定義

T = Propagation( E ⇒ σ_E ⇒ E′ ),   d(E′, E) < ε

時間とは、結果が履歴を選び、その履歴が次の結果を生む閉路の伝播である。

時間は原始的な線ではなく、結果—履歴—結果の閉路 closure の伝播として読まれる。

§C. 公理 v0.3

公理1 — 非可換性の実在

q₁q₂ ≠ q₂q₁

順序は表記ではなく演算実在。

公理2 — 順序の非絶対先在

非可換系で σ ⇒ E は成立する。だが σ ⇒ E だけでは系は閉じない。E ⇒ σ_E を加えて初めて整合閉路をなす。順序は原因側に固定されず、結果点から圧力下で事後選択されうる。

公理3 — 許容順序集合

結果駆動の選択は許容集合の上でのみ起きる。

Σ_Q^adm = { σ_i(Q) | σ_i は P, R, C の下で許容 }

公理4 — 圧力媒介

圧力は順序選択を曲げる。

Select(E, Σ_Q^adm; P₁) ≠ Select(E, Σ_Q^adm; P₂)

ただし、圧力 P の作用は許容集合 Σ_Q^adm の境界画定にある。 P は σ_E の値を直接指定しない。 P は「どの履歴を候補に入れ、どれを捏造として排すか」の線引きを行う。これにより P は何でも正当化する自由パラメータに堕ちることなく、複数の境界条件の連立として働く。

公理5 — 選択基準

σ_E = argmin_{σ ∈ Σ_Q^adm(P,R,C)} D( Eval_σ(Q), E )

P は候補集合を切り、D は候補内で不一致を測る。P は D に入らない。複数解が等価に最小を与えるとき、選択は同値類 [σ_E] を切り出す。

公理6 — 整合閉路と残差

E ⇒ σ_E ⇒ E′,   e* := d(E′, E)

e* < ε で閉路は固定。e* ≥ ε で棘が顕在化し次の結果へ射出。

公理7 — 物体安定

O_x = StableLoop( E_x, [σ_{E_x}]; P_x )

物体は同値類の局所安定。退化が安定性の源。

公理8 — 時間伝播

T = Propagation( E ⇒ σ_E ⇒ E′ )

10. 最短定義集

自由数 — 結果から順序を選ぶ数である。

自由四元数 — 観測された回転結果から、その結果を成立させる非可換な回転履歴を、圧力下で事後選択し、再評価で整合閉路を閉じる構造である。

反転 — 非可換演算系において、結果から履歴順序を選び直し、その履歴が再び結果を再現することで確定する閉路である。

残差 / 棘 — 整合閉路が結果を再現しきれなかった閉じ残し。許容を超えて顕在化したものが棘であり、閉路を次の結果へ送る。

物体 — 可能順序の衝突が、結果—履歴—結果の閉路として局所安定した同値類である。

時間 — 結果が履歴を選び、その履歴が次の結果を生む閉路の伝播である。

11. 一行要約

自由数とは、四元数の非可換性を足場に、結果が圧力下で許容可能な回転履歴を事後選択し、その整合閉路の閉じ残しを棘、安定した同値類を物体、伝播を時間として記述する、反転の数である。
